## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/woabu0/mshstudio.git
   ```
2. Start the application:
   ```bash
   Go Live
   ```
3. Open your browser and navigate to:
   ```
   http://localhost:5500
   ```
